# Pricing Tiers

Detailed comparison of Free and Pro tiers.

## Tier Comparison

| Feature | Free | Pro |
|---------|------|-----|
| **Price** | $0 | $4.99/month or $49/year (17% savings) |
| **Core Features** | | |
| Browse & watch clips | ✅ | ✅ |
| Vote & comment | ✅ | ✅ |
| Basic search | ✅ | ✅ |
| Submit clips | 10/day | 50/day |
| **Organization** | | |
| Favorites | 50 max | Unlimited |
| Collections | ❌ | ✅ Unlimited |
| Custom tags | ❌ | ✅ |
| **Search** | | |
| Keyword search | ✅ | ✅ |
| Advanced filters | ❌ | ✅ (date, views, duration) |
| Saved searches | ❌ | ✅ |
| **Experience** | | |
| Ads | Shown | Ad-free |
| Support | Community | Priority email |
| Data export | ❌ | ✅ CSV/JSON |
| Early access | ❌ | ✅ Beta features |

## Billing

- **Monthly**: $4.99 billed monthly
- **Yearly**: $49 billed annually (saves $10.88/year)

Payment via Stripe. Manage subscription in Customer Portal.

## Quotas

### Free Tier
- Clip submissions: 10/day
- Favorites: 50 total
- Collections: 0
- Search history: 10 recent (local)

### Pro Tier
- Clip submissions: 50/day
- Favorites: Unlimited
- Collections: Unlimited
- Search history: Unlimited (saved)

## Future Tiers (Planned)

- **Creator** ($19/month): Analytics, custom branding, API access
- **Team** ($99/month): Shared workspaces, collaboration, SSO
- **Enterprise**: Custom pricing, SLA, dedicated support
