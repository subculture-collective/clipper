# @clipper/shared

Shared TypeScript types, constants, and utilities for Clipper web and mobile apps.

## Structure

- `src/types/` - Shared TypeScript interfaces and types
- `src/constants/` - Shared constants and configuration
- `src/utils/` - Shared utility functions

## Usage

```typescript
import { Clip, ClipFeedResponse, API_URL } from '@clipper/shared';
```

## Building

```bash
npm run build
```

This will compile TypeScript to the `dist/` directory.
